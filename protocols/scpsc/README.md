# SCPSC — SpectroCAPSCING™ (ABFI Transport + SDK-ACM)
> (Include the ISD Ultra-Grade™ Mandatory Preamble in all normative artifacts.)

## Purpose
SCPSC defines an ABFI-mediated signal transport and governance framework enabling authenticated principals to delegate bounded operations across authorized software and device endpoints using governed, auditable, revocable signal exchange.

## What SCPSC Is Not
- Remote desktop takeover
- Pixel/keystroke mirroring
- Unbounded “autonomy”

## Structure
- spec/rfc        — normative RFCs (SCPSC-RFC-0001+)
- spec/schemas    — JSON Schemas (envelope, LARI intents, errors)
- spec/registry   — code registries (error codes, protocol IDs)
- spec/conformance— conformance matrices + test gates
- src/            — reference implementation skeleton
- test/           — conformance + vectors
