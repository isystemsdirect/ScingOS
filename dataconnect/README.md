# Firebase Data Connect (Scingular AI)

This folder is the local Data Connect project directory that the VS Code **Firebase Data Connect** extension detects.

- Config: `dataconnect/dataconnect.yaml`
- Schema: `dataconnect/schema/*.gql`
- Example connector + operations: `dataconnect/example/`
