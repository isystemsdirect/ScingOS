## Core Philosophy: Bona Fide Intelligence (BFI)

At the heart of ScingOS is **Bona Fide Intelligence**:

- **AI = Augmented Intelligence**, not Artificial Intelligence
- Systems **genuinely augment human capability** with transparent, trustworthy technology
- Intelligence amplifies perception and decision-making while **preserving human agency**
- All systems are authentic, human-serving, and transparent by design

## Technology Stack

- **Frontend**: Next.js / React / TypeScript
- **Backend**: Firebase (Auth, Firestore, Cloud Functions, Storage)
- **AI Platform**: SCINGULAR AI cloud services
- **Security**: BANE governance layer with cryptographic verification
- **Protocols**: AIP (real-time), ISDC (inspection data)
- **Native Shell**: Tauri (for desktop applications)
