# ScingGPT Icons

Place app icons here:

- `icon.ico` - Windows icon (256x256)
- `icon.icns` - macOS icon
- `icon.png` - Linux icon (512x512)

For multi-resolution icons, create a folder with PNG files:
- `16x16.png`
- `32x32.png`
- `48x48.png`
- `64x64.png`
- `128x128.png`
- `256x256.png`
- `512x512.png`

Use a tool like `electron-icon-maker` to generate all formats from a single source image.
